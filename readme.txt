== Description ==

<p>数以万计的分享，源自一个简单的按钮， <a href="http://fairyfish.net/">bShare 分享</a> 是一个强大的网页分享插件工具，您的读者可以将您网站上精采的内容快速分享、转贴到社群网络上。</p>

== Installation ==

1. 上传插件到 `/wp-content/plugins/` 目录
1. 在 WordPress 后台插件菜单下激活该插件
1. 该插件自动在在日志和静态页面正文后添加 bShare 按钮，如果你想在其他页面添加按钮，你可以使用 Widget 来添加。
